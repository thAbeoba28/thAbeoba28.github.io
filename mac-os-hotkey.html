Useful Mac Hotkeys

Command + A
Selects everything on the page

Command + C
Copy selected text and items

Command + V
Pastes copied text and/or items

Command + X
Copy and deletes selected text and items

Command + F
Uses search bar for the page to find any combination of characters

Command + T
Opens a new tab

Command + W
Closes tab that you are in

Command + Shift + T
Opens most recently closed tab

Command + R
Refresh the tab/page you are on

Option + Command + Esc
Force Quits

Command + Z
Undo previous action

Command + Shift + Z
Redo (Use after Command Z)

Shift + Command + 4
Takes a screenshot of selected area of the screen

Command + Option + W
Closes all tabs
