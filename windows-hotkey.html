Useful Windows Hotkeys


Meanings of words
  • Ctrl: Control
  • F[number]: Represents a single key, not pressing F and a number together
  • Windows button: May also look like a window icon

Ctrl + F
Uses search bar for the page to find any combination of characters

Ctrl + T
Opens a new tab

Ctrl + Shift + T
Opens most recently closed tab

Ctrl + W
Closes tab that you are in

Ctrl + Shift + W
Closes all tabs

Ctrl + W
Closes tab that you are in

Ctrl + Z
Undo previous action

Ctrl + Y
Redo (Use after Ctrl Z)

Ctrl + R
Refresh the tab/page you are on

Alt + F4
Force Quits

Windows + V
Displays recently copied items

Ctrl + A
Selects everything on the page

Ctrl + C
Copy selected text and items

Ctrl + V
Pastes copied text and/or items

Ctrl + A
Copy and deletes selected text and items

Windows + Shift + S
Takes a screenshot of selected area of the screen
